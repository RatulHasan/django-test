To Create virtualenv
```
virtualenv -p python3 .
```

To Activate virtualenv
```
source bin/activate
```

```
mkdir src && cd src
```

To Activate virtualenv
```
sudo pip install Django==1.11

django-admin startproject django-test .

```
python manage.py runserver

```
python manage.py migrate
